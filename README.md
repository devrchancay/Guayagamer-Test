## GuayaGamer

Requisitos

- Nodejs (v8.11.1)
- MongoDB (v3.6.5)

Para iniciar el proyecto.

- clonar repositorio.
- Instalar lerna `npm install -g lerna`.
- Instalar las dependencias `lerna bootstrap`
- Iniciar servidor `npm run server:dev`
- Iniciar aplicacion cliente `npm run client:dev`
